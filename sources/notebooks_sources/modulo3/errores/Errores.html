Al ver respuesta temporal, pudimos lograr la descripci√≥n de la respuesta transitoria de la se√±al de salida de un sistema, para ello definimos algunos par√°metros que identifican dicha respuesta, como ser \(Ts\), \(Tr\), \(Tp\) y \(Mp\). Ahora, nos centraremos en la ¬´parte estacionaria¬ª de la respuesta o salida del sistema, en otras palabras, nos interesa el valor final de la se√±al de salida luego del transitorio. Para esto, usaremos b√°sicamente dos cosas, por un lado una definici√≥n para el error de estado estacionario y por otro lado el teorema del valor final para el c√°lculo de dicho error.

Definici√≥n importante¬∂

Se define la se√±al de error \(E(s)\) el equivalente en el dominio de Laplace de la se√±al \(e(t)\) como:

\[ E(s)= R(s)-Y(s) \]

para todo sistema donde, \(R(s)\) es lo que quiero, es la referencia del sistema o el valor deseado y \(Y(s)\) es lo que √©s, es la salida del sistema o la medici√≥n.

Repaso del uso del Teorema del Valor Final (TVF)¬∂

Si todos los polos de \(sY(s)\) est√°n en el lado izquierdo de plano \(s\), entonces:

\[\lim_{t\rightarrow\infty}y(t)=\lim_{s\rightarrow 0}sY(s)\]

Ejemplo 1. Uso del teorema¬∂

Encontrar el valor final de la se√±al \(y(t)\) cuya transformada de Laplace es \(Y(s)=\dfrac{3(s+2)}{s(s^2+2s+10))}\).

Par aver si podemos aplicar el teorema de valor final debemos fijarnos que todos los polos de \(sY(s)\) est√©n en el lado izquierdo de plano \(s\).

Esto los podemos hacer:

Podemos ver que cumple con las hip√≥tesis del teorema del valor final, entonce resolviendo el l√≠mite tenemos que:

\[y(\infty)==\left. sY(s) \right\rvert_{s=0}=\frac{3\cdot 2}{10}=0.6\]

Podemos verificar haciendo la respuesta al impulso, grafic√°ndola y viendo el valor final al cual tiende.

Error de estado estacionario y sistema tipo¬∂

El error \(e(t)\) de un sistema lo definido como la referencia menos la salida, es decir:

\[e(t)=r(t)-y(t)\]

o lo que es lo mismo, en el dominio de Laplace:

\[E(s)=R(s)-Y(s)\]

En la figura mostramos dos diagramas gen√©ricos de bloques equivalentes, con la definici√≥n del error. Observemos del segundo diagrama de bloques, que si \(H/I = 1, e(t) = a(t)\).

Dicho error tendr√° un transitorio, y luego llegar√° a un estacionario al cual se lo denomina error de estado estacionario.

Figura 36 Diagrama gen√©rico para errores en estado estacionario¬∂

Caso especial: Realimentaci√≥n unitaria y constantes de error¬∂

Si \(H = I\), podemos modelar el sistema con realimentaci√≥n unitaria como muestra la siguiente figura.

Figura 37 Diagrama particular con \(H(s)\)=1 y \(G(s) = I(s)D(s)G_p(s)\).¬∂

Entonces tendremos:

\[F(s) = \dfrac{E(s)}{R(s)} = \dfrac{1}{1+G(s)}\]

Consideraremos a las entradas \(r(t)\) polin√≥micas de la forma

\[r(t) = \dfrac{t^k}{k!} \text{ para } t>0\]

siendo su transformada de Laplace

\[R(s)=\dfrac{1}{s^{k+1}}\]

Se toma como base un sistema mec√°nico gen√©rico como referencia para la nomenclatura, llamando para las entradas con \(k=0\) entrada de posici√≥n, para \(k=1\) entrada de velocidad, y para \(k=2\) entrada de aceleraci√≥n, sin tener en cuenta las unidades de la se√±al de entrada actual. Aplicando el teorema del valor final

\[\begin{split} \begin{align} \lim_{t\rightarrow \infty} e(t) & = e_{ss}= \lim_{s\rightarrow 0} sE(s)\\ & = \lim_{s\rightarrow 0} \frac{1}{1+G}R(s)\\ & = \lim_{s\rightarrow 0} s \frac{1}{1+G}\frac{1}{s^{k+1}} \end{align} \end{split}\]

Consideremos primero un sistema para donde \(G(s)\) no tiene polo en el origin, es decir no un ning√∫n integrador en \(G(s)\) y que se tiene una entrada escal√≥n \(R(s)=\dfrac{1}{s}\). De esta manera \(r(t)\) es una entrada polinomial de grado 0. En este caso podemos escribir las ecuaciones anteriores:

\[\begin{split} \begin{align} e_{ss} & = \lim_{s\rightarrow 0} s\frac{1}{1+G(s)}\frac{1}{s}\\ \dfrac{e_{ss}}{r_{ss}} & = \dfrac{e_{ss}}{1}=e_{ss} = \frac{1}{1+G(0)} \end{align} \end{split}\]

donde \(r_{ss} = \lim_{t\rightarrow \infty} =1 \).

Se define que este sistema es de Tipo 0 y definimos la constante \(G(0)\triangleq K_p\) que llamamos constantes de error de posici√≥n.

Notar que la ecuaci√≥n anterior brinda el error relativo, es decir la relaci√≥n entre el valor final del error y la y referencia.

Si la entrada fuera un polinomio mayor a 1, el error resultante crecer√≠a sin l√≠mites. Una entrada polin√≥mica de grado cero es el grado m√°s alto que un sistema de Tipo 0 puede seguir con error f√≠nito.

Si \(G(s)\) tiene un polo en el origen, entonces podr√≠amos continuar esta linea argumentativa y considerar un polinomio de orden 1 como entrada \(r(s)\), que es directo evaluarlo directamente de manera general. Para este caso es necesario describir el comportamiento de la planta y el controlador cuando \(s\) se aproxima a 0. Para este prop√≥sito agrupamos todos los t√©rminos excepto los polos en el origen de la funci√≥n \(G(s)\) en la funci√≥n \(G_0(s)\), de manera que podamos definir \(G_0(0)=K_n\) y podemos escribir la funci√≥n \(G(s)\) como:

\[G(s) = \frac{G_0(s)}{s^n}\]

De estar forma, si por ejemplo \(G(s)\) no tienen integradores, entonces \(n=0\); si tiene 1 integrador entonces \(n=1\), y as√≠ sucesivamente. Sustituyendo esto en la expresi√≥n del error:

\[\begin{split} \begin{align} e_{ss} & = \lim_{s\rightarrow 0 } s\frac{1}{1+\frac{G_0(s)}{s^n}}\frac{1}{s^{k+1}}\\ & = \lim_{s\rightarrow 0} \frac{s^n}{s^n+k_n}\frac{1}{s^k} \end{align} \end{split}\]

Para esta ecuaci√≥n podemos ver que si \(n> k\) entonces \(e =0\) y si \(n<k\) entonces el error \(e(t) \rightarrow \infty\).

Si \(n = k = 0\) entonces \(e_{ss} = \dfrac{1}{1+K_0}\) y si \(n = k \neq 0\) entonces \(e_{ss} = \frac{1}{K_n}\).

Como se dijo anteriormente si \(n= k =0\), la entrada es una se√±al polinomial de orden 0 y la constante \(K_p\) se llama constante de posici√≥n, se la escribe \(K_p\) y el sistema se cl√°sifica como sistema de Tipo 0.

Si \(n=k=1\), la entrada es un polinomio de orden 1 y se conoce como rampa o entrada de velocidad, y la constante \(K_1\) se la llama constante de velocidad y se escribe \(K_v\). Este sistema se clasifica como sistema de Tipo 1. De manera similar podemos extendernos a sistemas de Tipo 2 y mayores.

Tipo de sistema para la regulaci√≥n y rechazo a las perturbaciones¬∂

Un sistema tambi√©n puede ser clasificado seg√∫n su habilidad para rechazar perturbaciones de forma polin√≥mica de manera an√°loga al esquema de clasificaci√≥n hecho para el seguimiento a referencias.

Supongamos ahora que tenemos un sistema de la forma que se ve en la siguiente figura (notar el cambio de notaci√≥n):

Figura 38 Sistema a lazo cerrado con perturbaciones¬∂

Si queremos analizar como se comporta el error del sistema frente a una perturbaci√≥n \(W(s)\), vamos a pensar que la referencia \(r(t)\) se mantiene en 0. Adem√°s para este curso no se va a tener en cuenta el ruido de medici√≥n \(v(t)\). As√≠ \(V(s)=0\) y \(R(s)=0\).

Por lo tanto, recordando que \(E(s) = R(s)-Y(s)\).

As√≠ tenemos que:

\[E(s)= -Y(s)\]

Nos interesa la funci√≥n transferencia:

\[\frac{E(s)}{W(s)}=\frac{-Y(s)}{W(s)} = -\frac{G(s)}{1+G(s)D_{cl}} = T_w(s)\]

debido a que la entrada es igual a 0, la salida es el error (con el signo cambiado). De manera an√°loga que para las entradas referencias, se dice que el sistema es Tipo 0 si para un perturbaci√≥n escal√≥n un sistema resulta con error constante distinto de 0 en estado estacionario; Tipo 1 si para una rampa el error resulta constante distinto de 0 y as√≠ sucesivamente. En general, siguiendo el mismo enfoque usando para las entradas referencias, asumimos que una constante \(n\) y una funci√≥n \(T_{o,w}(s)\) puede ser definida con las propiedades que:

  • \(T_{o,w}(0) = \dfrac{1}{K_{n,w}}\)

  • \(T_w(s) = s^nT_{o,w}(s)\)

Entonces el error en estado estacionario a una entrada perturbaci√≥n que sea polinomial de grado \(k\) es:

\[\begin{split} \begin{align} y_{ss} & = \lim_{s\rightarrow 0 } \left[ sT_w(s)\frac{1}{s^{k+1}}\right]\\ & = \lim_{s\rightarrow 0} \left[ T_{o,w}(s)\frac{s^n}{s^{k}}\right] \end{align} \end{split}\]

De ac√° tenemos que se \(n>k\) el error es 0 y si \(n<k\) el error no es acotado. Si \(n = k\) el sistema es de tipo \(k\) y el error es dado por \(\dfrac{1}{K_{n,w}}\).

Ejemplo 3: Tipo de sistema para un motor de corriente continua con control de posici√≥n¬∂

Considerar el modelo simplificado del motor de corriente continua con realimentacion unitaria como se muestra en la figura:

Figura 39 Motor simplificado a lazo cerrado con perturbaciones¬∂

Analizar el tipo de sistema para las referencias \(R(s)\) y para las perturbaciones \(W(w)\) para el sistema de la figura anterior. Suponiendo que \(A\) y \(B\) son constantes y que:

  1. \(Dc(s) = k_p\)

  2. \(Dc(s) = k_p + \dfrac{k_i}{s}\)

La funci√≥n transferencia de la planta \(G(s)\) es:

\[G(s)=\frac{A}{s(\tau s + 1)}\]

Soluci√≥n Para el caso donde \(D_c(s)=k_p\).

Primero analizamos el sistema para los cambios en la referencia.

Tenemos \(G_0(s)= D_c(s)G(s) = k_p \dfrac{A}{s(\tau s + 1)}\).

Vemos que el sistema tiene un polo en 0 por lo que \(n=1\). Por lo tanto, para el seguimiento a referencia tenemos que el sistema es Tipo 1, y que tenemos que \(K_v\) es igual a

\[K_v=\lim_{s\rightarrow 0}s \dfrac{k_pA}{s(\tau s + 1)} = k_p A\]

y el error ser√≠a:

\[e_{ss}=\frac{1}{K_v}=\frac{1}{k_pA}\]

Analicemos ahora el caso 1 para el rechazo a perturbaciones.

Podemos ver que si hacemos \(R(s)=0\) tenemos que:

\[\begin{split} \begin{align} T_w(s) & = \dfrac{B}{A} \dfrac{\dfrac{A}{s(\tau s +1)}}{1+\dfrac{A}{s(\tau s +1)}k_p}= \dfrac{B}{s(\tau s+1)+Ak_p}\\ & = s^0 T_{0,w},\\ n & = 0,\\ K_{0, w} & = -\dfrac{Ak_p}{B}\\ \end{align} \end{split}\]

Entonces para rechazo a perturbaciones el sistema ser√≠a Tipo 0, y el error ser√°:

\[e_{ss} = -\frac{B}{Ak_p}\]

Ahora empecemos con el caso 2.

Primero analizamos el sistema para los cambios en la referencia.

Tenemos

\[G_0(s)= D_c(s)G(s) = \left(k_p + \dfrac{k_i}{s}\right) \dfrac{A}{s(\tau s + 1)} = \dfrac{sk_p + k_i}{s} \dfrac{A}{s(\tau s + 1)} = \dfrac{A(sk_p + k_i)}{s^2(\tau s + 1)}\]

Vemos que el sistema tiene dos polos en 0 por lo que \(n=2\). Por lo tanto, para el seguimiento a referencia tenemos que el sistema es Tipo 2, y que tenemos que \(K_a\) es igual a

\[K_a=\lim_{s^2\rightarrow 0}s^2 \dfrac{(sk_p+k_i)A}{s^2(\tau s + 1)} = k_i A\]

y el error ser√≠a:

\[e_{ss}=\frac{1}{K_a}=\frac{1}{k_iA}\]

Para rechazo a perturbaciones ser√≠a:

\[\begin{split} \begin{align} T_w(s) & = \dfrac{B}{A} \dfrac{\dfrac{A}{s(\tau s +1)}}{1+\dfrac{A}{s(\tau s +1)}\left(k_p + \dfrac{k_i}{s}\right)}\\ & = \dfrac{B}{s(\tau s+1)+A\left(\dfrac{k_p + k_is}{s}\right)}\\ & = \dfrac{Bs}{s^2(\tau s +1) + (k_ps+k_i)A }\\ & = s^1 T_{1,w},\\ n & = 1,\\ K_{1, w} & = -\frac{Ak_I}{B}\\ \end{align} \end{split}\]

Entonces para rechazo a perturbaciones es Tipo 1, y el error de estado estacionario a la rampa resulta:

\[e_{ss} = - \frac{B}{Ak_i}\]
