Sistema de segundo orden dominante¬∂

Una forma de seleccionar la ubicaci√≥n de los polos es hacer lo mismo que se hac√≠a en control cl√°sico, es decir buscar que el sistema sea parecido a un segundo orden dominante.

Esto es perfectamente factible en control moderno. Sin embargo, tiene la desventaja que no sabemos para nada que suceder√° con el esfuerzo de control \(u\). Ubicando los polos de esta manera podemos obtener esfuerzos de control imposibles de realizar por nuestros actuadores, o que lleven a consumos de energ√≠as demasiado altos para nuestro sistema.

Por esto, lo m√°s recomendable es usar t√©cnicas de control √≥ptimo para seleccionar los polos del sistema a lazo cerrado.

Lugar sim√©trico de las ra√≠ces¬∂

Una t√©cnica de control √≥ptimo ampliamente usado es el control regulador cuadr√°tico lineal (LQR). En este curso veremos una versi√≥n simplificada que se conoce como Lugar de las Ra√≠ces Sim√©trico. Esta versi√≥n simplificada consiste en resolver el problema de minimizaci√≥n del siguiente √≠ndice de costo:

\[\mathcal J = \int_0^{\infty}\left[\rho z^2(t)+u^2(t)\right]dt\]

donde:

\[\begin{split} \mathbf{\dot x} = \mathbf{Ax}+\mathbf B u\\ z = \mathbf{C_1x}\\ \end{split}\]

mediante la ley de control:

\[ u =-\mathbf{Kx}\]

Se puede demostrar que la ubicaci√≥n de los polos a lazo cerrado que minimizan el funcional de costo anterior, son los polos estables del lugar sim√©trico de las ra√≠ces dado por la ecuaci√≥n:

\[1+\rho G_0(-s)G_0(s)=0\]

donde \(G_0(s)\) es:

\[G_0(s) = \frac{Z(s)}{U(s)}=\mathbf{C_1}(s\mathbf I -\mathbf A)^{-1}\mathbf B\]

Por lo tanto el problema lo podemos resolver para todo \(\rho\) haciendo un lugar sim√©trico de las ra√≠ces de \(G_0(-s)G_0(s)\). El lugar dar√° sim√©trico respecto del eje \(j\omega\) ya que los polos y ceros de \(G_0(-s)G_0(s)\) tambi√©n estar√°n ubicados de forma sim√©trica.

def conjugate_tf(G):
    num = ctrl.tf(G).num[0][0]
    den = ctrl.tf(G).den[0][0]
    nume = [num[i]*((-1)**(len(num)%2+1-i)) for i in range(0, len(num))]
    dene = [den[i]*((-1)**(len(den)%2+1-i)) for i in range(0, len(den))]
    return ctrl.tf(nume,dene)
